# Installation

```sh
hochob@server:~$ sudo apt install python3-setuptools
hochob@server:~$ sudo apt install python3-pip
```

```sh
hochob@server:~$ sudo pip3 install --upgrade pip
```

```sh
hochob@server:~$ sudo pip3 install homeassistant
```
