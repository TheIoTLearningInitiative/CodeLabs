## Custom Component Hello State

```sh
user@server:~$ mkdir ~/.homeassistant/custom_components/
```

```sh
user@server:~$ nano ~/.homeassistant/custom_components/hello_state.py
```

```python
DOMAIN = 'hello_state'

def setup(hass, config):
    hass.states.set('hello.world', 'Paulus')

    return True
```
